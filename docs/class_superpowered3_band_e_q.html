Detailed Description

It doesn't allocate any internal buffers and needs just a few bytes of memory.

Parameters
lowDecibelThe decibel value of the low band. Read only.
midDecibelThe decibel value of the mid band. Read only.
highDecibelThe decibel value of the high band. Read only.
lowDecibelRoundedThe rounded decibel value of the low band. Read only.
midDecibelRoundedThe rounded decibel value of the mid band. Read only.
highDecibelRoundedThe rounded decibel value of the high band. Read only.
lowGainLow band gain. Read-write. See the adjust() method.
midGainMid band gain. Read-write. See the adjust() method.
highGainHigh band gain. Read-write. See the adjust() method.
lowGainModLow band gain modifier for advanced usage. Effective low gain is lowGain * lowGainMod. See the adjust() method.
midGainModMid band gain modifier for advanced usage. Effective mid gain is midGain * midGainMod. See the adjust() method.
highGainModHigh band gain modifier for advanced usage. Effective high gain is highGain * highGainMod. See the adjust() method.
lowKillIf true, the low band will be killed regardless the other variables. See the adjust() method.
midKillIf true, the mid band will be killed regardless the other variables. See the adjust() method.
highKillIf true, the high band will be killed regardless the other variables. See the adjust() method.
enabledTrue if the effect is enabled (processing audio). Read only. Use the enable() method to set.
